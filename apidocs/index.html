    Introduction

    Welcome to the PostIt Messaging API!

    PostIt is a modern group messaging application that allows users to communicate via groups. The application allows users to create account, create groups and add other users to groups. Group members can communicate by posting priority messages, members get email notification when higher priority messages are posted to the group.

    Authentication Endpoints

    Signup a user

    Sample Response

    
  {
    "message": "Signup successful.",
    "username": "johndoe" ,
    "email": "johndoe@gmail.com"
  }

    This endpoint handles user registration

    Endpoint

    '[POST:/api/user/signup]'

    Response

    • status: '201'
    • body: 'application/json'

    Query Parameters

    Parameter Status Description
    username Required Username of the PostIt user.
    email Required Email of the PostIt user.
    password Required Password of the PostIt user.
    Username and email are unique

    Errors

    The Posit Messaging API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is wrong.
    401 Unauthorized -- Your JWT key is wrong.
    403 Forbidden -- The requested is hidden for administrators only.
    404 Not Found -- The user/message/group could not be found.
    406 Not Acceptable -- You requested a format that isn't json.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
